# tmux cheatsheet

## Alt (prefix-free)

| Key | Action |
|-----|--------|
| `M-h` / `M-l` | Prev / next window |
| `M-Left` / `M-Right` | Swap window left / right |
| `M-j` / `M-k` | Next / prev session |
| `M-s` | Session chooser |
| `M-n` | New named session |
| `M-c` | New window |
| `M-\` | Horizontal split |
| `M--` | Vertical split |
| `M-x` | Kill pane |
| `M-z` | Zoom pane |
| `M-=` | Rebalance panes |
| `M-v` | Copy mode |
| `M-g` | Lazygit popup |
| `M-C` | Fork claude session |
| `M-?` | This cheatsheet |

## Prefix (`Ctrl+A`)

| Key | Action |
|-----|--------|
| `r` | Reload config |
| `]` | Paste from clipboard |
| `h/j/k/l` | Select pane |
| `←↓↑→` | Resize pane |
| `X` | Stop / kill session |

## Copy mode (vi)

| Key | Action |
|-----|--------|
| `v` | Begin selection |
| `y` | Copy |
| `Enter` | Copy & cancel |
